# Integration Guide

This guide will walk you through integrating Unpod's telephony and messaging services into your application.

## Prerequisites

- Node.js 16+ or Python 3.8+
- An active Unpod account
- API key with appropriate permissions

## Installation

### JavaScript/Node.js

```bash
npm install @unpod/sdk
# or
yarn add @unpod/sdk
```

### Python

```bash
pip install unpod-sdk
```

## Basic Usage

### Initialize the SDK

```javascript
// JavaScript
import { Unpod } from '@unpod/sdk';

const unpod = new Unpod({
  apiKey: 'your-api-key',
  environment: 'production' // or 'sandbox'
});
```

```python
# Python
from unpod import Unpod

unpod = Unpod(
    api_key="your-api-key",
    environment="production"  # or "sandbox"
)
```

## Making Your First Call

```javascript
// JavaScript
const call = await unpod.calls.create({
  to: '+14155551234',
  from: '+14155556789',
  answerUrl: 'https://your-app.com/answer',
  timeout: 30
});

console.log(`Call SID: ${call.sid}`);
```

## Sending an SMS

```javascript
// JavaScript
const message = await unpod.messages.create({
  to: '+14155551234',
  from: '+14155556789',
  body: 'Hello from Unpod!',
  statusCallback: 'https://your-app.com/status'
});

console.log(`Message SID: ${message.sid}`);
```

## Handling Webhooks

Unpod will send HTTP requests to your webhook URLs for call and message events.

### Example Webhook Handler (Node.js)

```javascript
import express from 'express';
import bodyParser from 'body-parser';

const app = express();
app.use(bodyParser.urlencoded({ extended: false }));

app.post('/webhook', (req, res) => {
  const event = req.body;
  
  switch (event.type) {
    case 'call.initiated':
      console.log('Call initiated:', event.callSid);
      break;
    case 'call.answered':
      console.log('Call answered:', event.callSid);
      break;
    case 'message.sent':
      console.log('Message sent:', event.messageSid);
      break;
    case 'message.delivered':
      console.log('Message delivered:', event.messageSid);
      break;
  }
  
  res.status(200).end();
});

app.listen(3000, () => {
  console.log('Webhook server running on port 3000');
});
```

## Testing Your Integration

   ```javascript
   const unpod = new Unpod({
     apiKey: 'your-test-api-key',
     environment: 'sandbox'

   ```

   ```bash
   ngrok http 3000
   ```
