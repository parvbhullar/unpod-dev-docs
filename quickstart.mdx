## Developer Quickstart

Get the Unpod platform running locally and deploy your first voice agent. Choose the setup method that works best for you.

### Prerequisites

Before you begin, ensure you have:

- **Node.js** v20+ / **npm** v10+
- **Python** 3.11+
- **Docker** and **Docker Compose**
- **Git**

---

### Option 1: One-Command Setup (Recommended)

The fastest way to get everything running:

```bash
git clone https://github.com/parvbhullar/unpod.git
cd unpod
make quick-start
```

This configures environment files, installs Node and Python dependencies, starts infrastructure services (PostgreSQL, MongoDB, Redis, Centrifugo), and runs database migrations.

Then launch the development servers:

```bash
make dev
```

---

### Option 2: Manual Setup

If you prefer full control over each step:

```bash
git clone https://github.com/parvbhullar/unpod.git
cd unpod
npm install
```

Set up the Python environment:

```bash
python3 -m venv apps/backend-core/.venv
source apps/backend-core/.venv/bin/activate
pip install -r apps/backend-core/requirements/local.txt
```

Start infrastructure services:

```bash
docker compose -f docker-compose.simple.yml up -d postgres mongodb redis centrifugo
```

Run migrations and seed data:

```bash
cd apps/backend-core
python manage.py migrate --no-input
python manage.py seed_reference_data
python manage.py setup_schedules
cd ../..
```

Start the development servers:

```bash
npm run dev
```

For detailed environment variable configuration, see the [Configuration](/configuration) page.

---

### Option 3: Docker Only

If you just want to run the platform without a local dev environment:

```bash
git clone https://github.com/parvbhullar/unpod.git
cd unpod
docker compose -f docker-compose.simple.yml up -d --build
```

Default credentials: `admin@unpod.dev` / `admin123`

---

### Access Points

Once running, the following services are available:

| Service | URL |
|---------|-----|
| Frontend | http://localhost:3000 |
| Backend API | http://localhost:8000/api/v1/ |
| Admin Panel | http://localhost:8000/unpod-admin/ |
| FastAPI Docs | http://localhost:9116/docs |
| Centrifugo Admin | http://localhost:8100 |

---

### Verify It Works

1. Open http://localhost:3000 in your browser
2. Log in with the default credentials or create a new account
3. Create a voice agent from the AI Studio
4. Assign a phone number and make a test call
