## What Is a Voice Profile?

A voice profile bundles a **Speech-to-Text (STT)** provider and a **Text-to-Speech (TTS)** provider into a single reusable configuration. When you attach a voice profile to an agent, every call that agent handles will use those providers automatically - including automatic failover if a primary provider goes down.

```
  Caller speaks
       │
       ▼  STT (e.g. Deepgram)
  User text
       │
       ▼  Your dialog machine
  Agent text
       │
       ▼  TTS (e.g. Cartesia)
  Synthesized audio
       │
       ▼
  Caller hears
```

---

## Supported Providers

### STT (Speech-to-Text)

| Provider | Key strengths |
|---|---|
| `deepgram` | Low latency, high accuracy, multilingual |
| `cartesia` | Cartesia combined STT/TTS stack |

### TTS (Text-to-Speech)

| Provider | Key strengths |
|---|---|
| `cartesia` | Ultra-low latency, natural voices |
| `elevenlabs` | High-quality, expressive, voice cloning |

## Listing Voice Profiles via SDK

```python
import asyncio
from unpod import AsyncClient

async def main():
    async with AsyncClient(api_key="sk-...") as client:
        # All profiles
        profiles = await client.voice_profiles.list()

        # Filter by language
        en_profiles = await client.voice_profiles.list(language="en")
        es_profiles = await client.voice_profiles.list(language="es")

        for p in profiles:
            print(
                p.id,
                p.name,
                f"gender={p.gender}",
                f"quality={p.quality}",
                f"language={p.language}",
            )

asyncio.run(main())
```

### VoiceProfile fields

| Field | Type | Description |
|---|---|---|
| `id` | `str` | Profile ID to use in agent creation (`vp_...`) |
| `name` | `str` | Human-readable name, e.g. `"Emma (HD)"` |
| `gender` | `str` | `male`, `female`, or `neutral` |
| `quality` | `str` | `standard` or `high` |
| `language` | `str` | BCP-47 language tag, e.g. `"en"`, `"es"`, `"de"` |
| `stt_provider` | `str` | Active STT provider |
| `tts_provider` | `str` | Active TTS provider |

## Getting a Single Profile

```python
import asyncio
from unpod import AsyncClient

async def main():
    async with AsyncClient(api_key="sk-...") as client:
        profile = await client.voice_profiles.get("vp_en_female_hd")
        print(profile.name, profile.stt_provider, profile.tts_provider)

asyncio.run(main())
```

## Choosing the Right Profile

### By use case

| Use case | Recommended profile qualities |
|---|---|
| Customer support | High quality, natural female or male, low latency |
| Sales outreach | Expressive, high quality, brand-matched voice |
| Appointment booking | Standard quality is fine - cost efficient |
| Multilingual | Match language to caller's region |

### By latency requirements

Voice latency has three components: STT transcription + LLM thinking + TTS synthesis. Profiles using Deepgram + Cartesia typically achieve the lowest total end-to-end latency.

```
Total latency = STT_ms + LLM_ms + TTS_ms

Deepgram STT:   ~150ms
Cartesia TTS:   ~80ms
Elevenlabs TTS: ~200ms
```

<Tip>
For latency-sensitive applications, choose profiles with `deepgram` STT and `cartesia` TTS. Check `profile.quality == "high"` only if voice naturalness matters more than speed.
</Tip>

## Using a Profile When Creating an Agent

Pass the profile `id` directly when creating or updating an agent:

```python
import asyncio
from unpod import AsyncClient

async def main():
    async with AsyncClient(api_key="sk-...") as client:
        # List to find the right ID
        profiles = await client.voice_profiles.list(language="en")
        hd_female = next(
            p for p in profiles if p.quality == "high" and p.gender == "female"
        )

        # Create agent with that profile
        agent = await client.agents.create(
            name="Support Bot",
            voice_profile=hd_female.id,
        )
        print("Created agent with profile:", hd_female.name)

asyncio.run(main())
```

## Switching a Profile on an Existing Agent

```python
import asyncio
from unpod import AsyncClient

async def main():
    async with AsyncClient(api_key="sk-...") as client:
        agent = await client.agents.update(
            "agt_...",
            voice_profile="vp_es_male_std",   # switch to Spanish male
        )
        print("Updated voice profile:", agent.voice_profile)

asyncio.run(main())
```
